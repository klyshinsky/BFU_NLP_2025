# BFU_NLP_2025
NLP practical course at I. Kant BFU.

Экспресс-Курс посвящен изучению основ машинного обучения (векторизация, классификация, кластеризация) и использования нейронных сетей. Курс не рассчитан на понимание происходящих процессов, а лишь на возможность их использования.

## Тетрадки к лекциям

[Лекция 2025.12.01, часть 1](https://github.com/klyshinsky/BFU_NLP_2025/blob/main/Lecture_1_Reduce_Space.ipynb) - Сокращение размерности пространства признаков.

[Лекция 2025.12.01, часть 2](https://github.com/klyshinsky/BFU_NLP_2025/blob/main/Lecture_2_W2V_Bert.ipynb) - Семантическое векторное пространство Word2Vec и Bert.

[Лекция 2025.12.08, часть 1](https://github.com/klyshinsky/BFU_NLP_2025/blob/main/Lecture_3_clustering.ipynb) - Кластеризация: k-средних, иерархическая, DBScan.

[Лекция 2025.12.08, часть 2](https://github.com/klyshinsky/BFU_NLP_2025/blob/main/Lecture_4_Classification.ipynb) - Классификация: k ближайших соседей, линейная регрессия.
